\section{C++ for TDD:}
\begin{itemize}
    \item {软件的工作目标:} 
    \begin{itemize}
        \item 输入电路，输出电路的TDD表示。
    \end{itemize}
    \item {运行环境：} 
    \begin{itemize}
        \item c++ standard 17
        \item cmake >= 3.20.0
        \item xtl >= 0.7.5
        \item xtensor >= 0.24.0
        \item graphiz >= 2.43.0
        \item xtensor-python >= 0.26.0
        \item pybind11 >= 2.12.0
        \item Numpy >= 1.20.0
    \end{itemize} 
    \item {适应的工作对象:}
    \begin{itemize}
        \item 小型电路
    \end{itemize}
    \item {测试样例:}
    \begin{itemize}
        \item 基础门，如CNOT,H,S,swap
        \item 随机生成的小规模电路
    \end{itemize}
    \item {自我测试实验报告:}
    \begin{itemize}
        \item 所有基础门正确转化为TDD表示
        \item 大部分小规模电路正确转换，部分出现了内存占用过大的问题。
    \end{itemize}
\end{itemize}
